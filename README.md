# 🏧 ATM Machine Console Application

A simple Java-based console ATM simulation built using OOP principles.

## 🚀 Features
- View Balance
- Withdraw Amount
- Deposit Amount
- View Mini Statement
- PIN Authentication

## 🛠 Technologies Used
- Java
- OOP (Interface, Implementation, Encapsulation)
- HashMap for transaction storage

## 📂 Project Structure
- AtmOperations (Interface)
- AtmOperationsImp (Implementation Class)
- ATM (Model Class)
- Main (Driver Class)

## 📌 Concepts Practiced
- Interfaces
- Method Overriding
- Polymorphism
- Collections (HashMap)
- Exception Handling Basics
